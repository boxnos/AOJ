Capitalize
==========

Write a program which replace all the lower-case letters of a given text
with the corresponding captital letters.

Input
-----

A text including lower-case letters, periods, and space.

Output
------

The converted text.

Sample Input
------------

    this is a pen.

Output for the Sample Input
---------------------------

    THIS IS A PEN.
