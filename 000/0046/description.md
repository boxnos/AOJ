Differential
============

今まで登ったことのある山の標高を記録したデータがあります。このデータを読み込んで、一番高い山と一番低い山の標高差を出力して終了するプログラムを作成してください。

Input
-----

``` {.exp}
山の高さ（実数）
...
...
```

Output
------

一番高い山と一番低い山の標高差（実数）。出力は0.01以下の誤差を含んでもよい。

Sample Input
------------

    3776.0
    1819.0
    645.2
    2004.1
    1208.6

Output for the Sample Input
---------------------------

    3130.8
