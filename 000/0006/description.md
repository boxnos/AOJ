Reverse Sequence
================

Write a program which reverses a given string *str*.

Input
-----

*str* (the size of *str* ≤ 20)

Output
------

Reversed *str*

Sample Input
------------

    w32nimda

Output for the Sample Input
---------------------------

    admin23w

* * * * *

Source: PC Koshien 2003 , All-Japan High School Programming Contest,
Aizu-Wakamatsu, Japan, 2003\
