Sort I
======

与えられたｎ個の数字を昇順に並び替えて出力するプログラムを作成せよ。

Input
-----

１行目にｎが与えられる。２行目にｎ個の数字が空白区切りで与えられる。

Output
------

昇順に整列したｎ個の数字を空白区切りで１行に出力せよ。

Constraints
-----------

ただし、ｎの値は1000以下と考えてよい。

Sample Input
------------

    5
    5 4 3 2 1

Sample Output
-------------

    1 2 3 4 5
