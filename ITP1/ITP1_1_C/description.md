Rectangle
=========

たて *a* cm よこ *b* cm
の長方形の面積と周の長さを求めるプログラムを作成して下さい。

Input
-----

*a* と *b* が１つの空白で区切られて与えられます。

Output
------

面積と周の長さを１つの空白で区切って１行に出力して下さい。

Sample Input
------------

    3 5

Sample Output
-------------

    15 16

Note
----

**[２つ以上の数値の入力 [C/C++]](javascript:void(0))**

scanf で２つの整数を続けて読み込むには

    scanf("%d %d", &a, &b);

と記述します。

cin で２つの整数を続けて読み込むには

    cin >> a >> b;

と記述します。
